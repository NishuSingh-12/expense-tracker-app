# Expense Tracker App

A simple expense tracker built using **React**, **Vite**, and **Tailwind CSS v4**.
This project is developed step by step to practice real-world React fundamentals
and maintain clean, feature-based Git commits.

## Features

- Add an expense with category, amount, and date
- Delete an expense
- Filter expenses by category
- Show total expenses and category-wise totals
- Data persistence using localStorage
- Responsive UI using Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS v4
- JavaScript (ES6)
- localStorage
